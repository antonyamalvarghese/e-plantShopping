# e-plantShopping

**e-plantShopping** is a React + Redux plant shopping cart application (“Paradise Nursery”) where users can browse plants by category, add items to a cart, update quantities, remove items, and view live totals.

## Features
- Browse plants by categories (Air Purifying, Aromatic, Medicinal, etc.)
- Add to Cart (Redux global state)
- Increase / decrease quantity
- Remove items from cart
- Live cart total amount + per-item subtotal
- Cart icon shows total quantity badge
- Responsive layout for navbar and product grid

## Tech Stack
- React (Vite)
- Redux Toolkit + React Redux
- CSS
